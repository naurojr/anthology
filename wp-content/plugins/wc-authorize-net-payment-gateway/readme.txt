WC Authorize.net Payment Gateway addon adds a payment option for customers to pay with their Credit Cards.

== Description ==

WC Authorize.net Payment Gateway addon adds a payment option for customers to pay with their Credit Cards.

<h3>WooCommerce 2.6x Compatible</h3>

== Features ==
1. Accept Card right on your website.
2. No redirecting on other url.
3. Easy to install and configure
4. Option to configure success & failure message
5. Safe way to process credit cards and debit cards on WooCommerce using Authorize.net
6. This plugin uses internal card processing, so faster and more relaible.


Support

So that others can share in the answer, please submit your support requests through the WordPress forums for WC Authorize.net Payment Gateway(http://wordpress.org/support/plugin/wc-authorizenet-aim-payment-gateway).

== Installation ==

Minimum requirements:    
    
Wordpress 4.5+ 
WooCommerce 2.6+   
PHP 5.x    
MySQL 4.x+  

Installation from within backend:

1. In plugin manager, click "Add New"
2. Search for "wc-authorizenet-aim-payment-gateway"
3. Click "install"
4. Activate the plugin once it is installed

FTP upload installation method:

1. Upload the `wc-authorizenet-aim-payment-gateway` folder to the `/wp-content/plugins/` directory using your FTP client
2. Activate the plugin through the 'Plugins' menu

Zip upload installation method (make sure uploads up to 5MB are allowed for your hosting):

1. Login to your WordPress site administrator panel and head over the 'Plugins' menu  
2. Click 'Add New'  
3. Choose the 'Upload' option
4. Click **Choose file** (**Browse**) and select the wc-authorizenet-aim-payment-gateway.*.zip file.   
5. Click **Install Now** button.    
6. Once it is complete, activate the plugin.
